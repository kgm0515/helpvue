# 简介

@helpvue/pvue 是一个基于 Vue3 开发的 PC 端组件库

特性

- 🚀 提供 1+ 个高质量通用组件
- 🚀 组件十分轻量
- 🛠️ 支持按需引入
- 💪 支持 SSR
- 💡 支持 Typescript
